# Mitech-cli
Mitech S.r.l. utility CLI

Questo pacchetto fornisce il comando **mitech**, utility ad uso interno di Mitech S.r.L.

## Prerequisiti
E' richiesto l'uso di Node 12 o superiore

## Installazione
```shell
git clone https://github.com/mitechsrl/mitech-cli.git
npm install
npm link
```
